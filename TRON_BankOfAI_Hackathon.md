---

## 📌 活动概览

本次 Hackathon 旨在推动开发者探索 **AI Agent × Web3 × TRON 生态** 的创新应用。

参赛者需基于 **TRON 生态**，结合 **Bank of AI 提供的 AI Agent 金融基础设施**，在两周内构建一个可公开访问的 Demo 产品。

**项目形式可以是：**
- AI Agent 工具
- AI 自动化服务
- AI 金融 / 支付应用
- AI 数据分析应用

**核心目标：构建能够在链上进行经济活动的 AI Agent 应用。**

## 💡 开发方向参考

### AI Payment Agent
AI Agent 提供服务并自动收取 USDT / USDD。  
*例如：AI Writing Assistant、AI Trading Signal Service*

### AI DeFi Assistant
AI Agent 分析 DeFi 收益并自动执行策略。  
*例如：Yield Optimizer、AI Portfolio Manager*

### AI On-chain Data Agent
AI Agent 分析 TRON 链上数据并提供洞察。  
*例如：TRON Whale Tracker、TRON DeFi Dashboard*

### AI Automation Agent
AI Agent 自动执行链上任务。  
*例如：自动空投领取、自动 NFT Mint、自动交易*

## 📋 参赛规则

### ① 开源要求
- 项目须在 **GitHub 开源**，包含完整 **README** 及 **commit 迭代记录**
- README 需包含：项目介绍、技术架构、Bank of AI 使用方式、Demo 使用说明
- 将项目代码提交到**共学小组自己的文件夹**中，并向主仓库发起 **Pull Request（PR）**

### ② 产品上线
- 项目须**部署上线**，可被公开访问（Web App / AI Agent API / Demo Interface）

### ③ 技术要求
- 必须使用 **TRON 网络**
- 必须集成 **Bank of AI 相关基础设施**
- 若涉及支付，需支持 **USDT** 或 **USDD**
